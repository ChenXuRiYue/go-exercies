# 概念
声明是语法层的行为。Go 语法中给定了相关的声明语句：
不同的声明方式，同时告知编译器声明对象中重要属性（适合在创建之初明确，并且只发生一次）。
比如描述该对象是哪种域（对象，函数，或类型）
是否可变
主要有以下几种：
1. var    变量声明 
2. const  常量声明
3. type   类型声明
4. func   函数实体声明
5. package 包声明

## var 类型声明
格式
1. var 变量名 数据类型 = 值


## const 声明
格式
1. const 常量名 数据类型 = 值

## 函数声明
格式
func 函数名(参数列表) 返回值类型 { 函数体 }
